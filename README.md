# Features
* You can add prayes that you forgot to do.
* Application will save the count of all the prayes you didn't do.

# Technologies & Open-source libraries
* Kotlin mainly, Coroutines
* RoomDatabase
* Architecture
  - MVVM (model view view-model).
  - Repository pattern.
